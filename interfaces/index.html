    Описание интерфейсов

    Общие элементы интерфейса

    Шапка сайта (Header)

    Шапка содержит:

    • Логотип/Название - "Racing Portal" (ссылка на главную)
    • Поиск - форма поиска гонок по названию
    • Навигация - кнопки входа/регистрации или информация о пользователе и выход

    Для неавторизованных пользователей: - Кнопка "Войти" (синяя) - Кнопка "Регистрация" (зеленая)

    Для авторизованных пользователей: - Приветствие с именем пользователя - Кнопка "Выйти" (красная)

    Боковое меню (Offcanvas)

    Мобильное меню с навигацией:

    • Главная
    • Табло победителей
    • Мой профиль (только для авторизованных)
    • Войти / Регистрация / Выйти

    Подвал (Footer)

    • Информация об авторских правах
    • Ссылки (About, Contact, Privacy)

    Основные страницы

    1. Главная страница - Список гонок (/)

    Описание: Отображает список всех доступных гонок с возможностью поиска и пагинации.

    Элементы интерфейса:

    • Заголовок - "Список гонок"
    • Форма поиска - поле ввода для поиска по названию гонки
    • Список гонок:
    • Название гонки (ссылка на детальную страницу)
    • Дата проведения гонки
    • Пагинация - навигация по страницам (если гонок больше 5)

    Функциональность: - Поиск гонок в реальном времени - Переход на детальную страницу гонки - Пагинация результатов

    2. Детальная страница гонки (/race/<id>/)

    Описание: Полная информация о гонке, включая таблицу результатов, регистрацию и комментарии.

    Элементы интерфейса:

    Блок информации о гонке

    • Название гонки - крупный заголовок
    • Описание - текст описания гонки
    • Дата и место - метаинформация

    Таблица результатов (Табло победителей)

    Таблица со следующими колонками:

    Колонка Описание
    Позиция Место участника в гонке
    ФИО участника Полное имя гонщика
    Название команды Команда участника
    Описание автомобиля Название и описание авто
    Описание участника Информация о гонщике
    Опыт (лет) Количество лет опыта
    Класс участника Категория (A, B, C)
    Время заезда (мс) Время финиша в миллисекундах

    Особенности: - Сортировка по позиции - Выделение позиций цветом - Отображение "—" для пустых полей

    Блок регистрации

    Для неавторизованных: - Ссылка "Войдите" для входа в систему

    Для авторизованных без регистрации: - Кнопка "Зарегистрироваться" (синяя)

    Для авторизованных с регистрацией: - Кнопка "Редактировать регистрацию" (желтая) - Кнопка "Удалить регистрацию" (красная) с подтверждением

    Блок комментариев

    • Список комментариев:
    • Имя комментатора
    • Дата и время
    • Рейтинг (1-10)
    • Тип комментария

    • Текст комментария

    • Форма добавления комментария:

    • Текстовое поле для комментария
    • Выбор типа комментария (выпадающий список)
    • Поле рейтинга (1-10)
    • Кнопка "Отправить"

    Примечание: Для добавления комментария требуется наличие профиля участника.

    3. Страница регистрации на гонку (/race/<id>/register/)

    Описание: Форма для регистрации участника на конкретную гонку.

    Элементы интерфейса:

    • Заголовок - "Регистрация на: [Название гонки]"
    • Форма регистрации:
    • Выбор автомобиля (выпадающий список)
    • Кнопка "Зарегистрироваться"
    • Кнопка "Отмена"

    Особенности: - Если у пользователя нет автомобилей, отображается предупреждение с предложением добавить автомобиль - Валидация уникальности регистрации

    4. Страница редактирования регистрации (/registration/<id>/edit/)

    Описание: Форма для изменения данных регистрации на гонку.

    Элементы интерфейса:

    • Заголовок - "Редактирование регистрации на: [Название гонки]"
    • Форма:
    • Выбор автомобиля (предзаполненный)
    • Кнопка "Сохранить изменения"
    • Кнопка "Отмена"

    Ограничения: - Доступно только для владельца регистрации

    5. Страница входа (/login/)

    Описание: Форма входа в систему.

    Элементы интерфейса:

    • Заголовок - "Вход в систему"
    • Форма входа:
    • Поле "Имя пользователя"
    • Поле "Пароль"
    • Кнопка "Войти"
    • Ссылка на регистрацию - "Нет аккаунта? Зарегистрироваться"

    Особенности: - Отображение ошибок валидации - Сохранение параметра next для редиректа после входа

    6. Страница регистрации пользователя (/register/)

    Описание: Форма создания нового аккаунта.

    Элементы интерфейса:

    • Заголовок - "Регистрация"
    • Форма регистрации:
    • Поле "Имя пользователя"
    • Поле "Пароль"
    • Поле "Подтверждение пароля"
    • Кнопка "Зарегистрироваться"
    • Ссылка на вход - "Уже есть аккаунт? Войти"

    Особенности: - Валидация пароля (минимальная длина, сложность) - Автоматическое создание профиля участника - Автоматический вход после регистрации

    7. Страница профиля (/profile/)

    Описание: Личный кабинет пользователя с информацией о профиле, автомобилях и регистрациях.

    Элементы интерфейса:

    Блок информации о профиле

    • ФИО
    • Название команды
    • Опыт (лет)
    • Класс участника
    • Описание участника
    • Кнопка "Редактировать"

    Блок автомобилей

    • Список автомобилей с названием и описанием
    • Кнопка "Добавить автомобиль"

    Блок регистраций

    • Список всех регистраций пользователя:
    • Название гонки (ссылка)
    • Дата гонки
    • Автомобиль
    • Позиция (если указана)
    • Кнопки "Редактировать" и "Удалить"

    8. Страница редактирования профиля (/profile/edit/)

    Описание: Форма изменения данных профиля участника.

    Элементы интерфейса:

    • Заголовок - "Редактирование профиля"
    • Форма:
    • ФИО
    • Название команды
    • Описание участника (многострочное)
    • Опыт (лет) - числовое поле
    • Класс участника (выпадающий список: A, B, C)
    • Кнопка "Сохранить"
    • Кнопка "Отмена"

    9. Страница добавления автомобиля (/car/add/)

    Описание: Форма для добавления нового автомобиля в профиль.

    Элементы интерфейса:

    • Заголовок - "Добавить автомобиль"
    • Форма:
    • Название автомобиля
    • Описание автомобиля (многострочное)
    • Кнопка "Добавить"
    • Кнопка "Отмена"

    10. Табло победителей (/leaderboard/)

    Описание: Обзорная страница с результатами всех гонок.

    Элементы интерфейса:

    • Заголовок - "Табло победителей автогонок"

    Для каждой гонки: - Название гонки (ссылка на детальную страницу) - Дата и место проведения - Количество участников

    Таблица топ-3 результатов: - Место - ФИО участника - Команда - Автомобиль - Время (мс) - Класс участника

    • Ссылка "Посмотреть все результаты →"

    Особенности: - Отображаются только гонки с опубликованными результатами - Показываются топ-3 участника каждой гонки

    Админ-панель (/admin/)

    Описание: Интерфейс Django Admin для управления данными.

    Разделы:

    1. ParticipantProfile - управление профилями участников
    2. Car - управление автомобилями
    3. Race - управление гонками
    4. Registration - управление регистрациями:
    5. Редактирование позиции и времени заезда прямо в списке
    6. Группировка полей для удобства
    7. Comment - управление комментариями

    Особенности: - Фильтры и поиск - Массовое редактирование - Удобный интерфейс для указания результатов гонок

    Дизайн и стилизация

    Цветовая схема

    • Основной цвет: Синий (#2563EB)
    • Успех: Зеленый (#16A34A)
    • Предупреждение: Желтый (#CA8A04)
    • Опасность: Красный (#DC2626)
    • Нейтральный: Серый (#6B7280)

    Компоненты

    • Использование Tailwind CSS для стилизации
    • Bootstrap 5 для компонентов (навигация, формы)
    • Адаптивный дизайн
    • Современный UI с тенями и скругленными углами

    Типографика

    • Заголовки: жирный шрифт, различные размеры
    • Основной текст: читаемый, достаточный контраст
    • Метаинформация: меньший размер, серый цвет
