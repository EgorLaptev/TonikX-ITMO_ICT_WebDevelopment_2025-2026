# Документация API системы управления отелем

Добро пожаловать в документацию API для системы управления отелем, разработанной с использованием Django REST Framework.

## Описание

Данная система предоставляет RESTful API для управления отелем, включая работу с номерами, гостями, сотрудниками, бронированиями и расписанием уборки.

## Основные возможности

- Управление типами номеров (Room Types)
- Управление этажами (Floors)
- Управление номерами (Rooms)
- Управление гостями (Guests)
- Управление проживаниями (Stays)
- Управление сотрудниками (Employees)
- Управление расписанием уборки (Cleaning Schedule)

## Базовый URL

Все API endpoints доступны по адресу:

```
http://localhost:8000/api/
```

## Аутентификация

API использует Token Authentication. Для получения токена необходимо пройти аутентификацию через endpoints Djoser.

Подробнее об аутентификации см. раздел [Аутентификация](authentication.md).
