# Инструкция по использованию документации

## Установка MkDocs

Для работы с документацией необходимо установить MkDocs и тему Material:

```bash
pip install -r requirements-docs.txt
```

или

```bash
pip install mkdocs mkdocs-material
```

## Запуск локального сервера документации

Для просмотра документации локально выполните:

```bash
mkdocs serve
```

Документация будет доступна по адресу: `http://127.0.0.1:8000`

## Сборка документации

Для создания статической версии документации выполните:

```bash
mkdocs build
```

Собранная документация будет находиться в папке `site/`.

## Развертывание на Read the Docs

1. Зарегистрируйтесь на [Read the Docs](https://readthedocs.org/)
2. Подключите ваш репозиторий
3. Укажите файл конфигурации: `mkdocs.yml`
4. Read the Docs автоматически соберет и опубликует документацию

## Структура документации

- `index.md` - главная страница
- `api/` - документация API endpoints
- `authentication.md` - описание аутентификации
- `models.md` - описание моделей данных
