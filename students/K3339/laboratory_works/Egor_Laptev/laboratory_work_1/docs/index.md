# Лабораторная работа 1. Работа с сокетами

## Общая информация

В данной работе реализованы несколько сетевых приложений на Python с использованием библиотеки `socket`.
Применяемые технологии:

* Python 3.x
* Библиотека `socket`
* Модуль `threading` для многопользовательского чата
* Модуль `urllib.parse` для обработки POST-запросов
* HTML и HTTP для веб-сервера

---

## Итоги

* Изучены основы работы с **TCP** и **UDP** сокетами.
* Реализованы клиент-серверные приложения для обмена сообщениями и математических вычислений.
* Создан веб-сервер для работы с формами и динамической таблицей оценок.
* Реализован многопользовательский чат с потоками.

**Использованные библиотеки:**
`socket`, `threading`, `urllib.parse`

**Заключение:**
Все приложения работают корректно и демонстрируют принципы работы TCP и UDP, обработку HTTP-запросов, многопоточность и динамическое отображение данных в HTML.
