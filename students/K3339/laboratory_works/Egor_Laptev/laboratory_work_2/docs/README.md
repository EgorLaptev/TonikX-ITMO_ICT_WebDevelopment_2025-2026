# Документация проекта

Эта директория содержит документацию проекта, созданную с помощью MkDocs.

## Просмотр документации

### Локальный запуск

1. Установите MkDocs Material:
```bash
pip install mkdocs-material
```

2. Запустите сервер документации:
```bash
mkdocs serve
```

3. Откройте в браузере: http://127.0.0.1:8001/

### Сборка статической версии

```bash
mkdocs build
```

Статические файлы будут созданы в директории `site/`.

## Структура документации

- `index.md` - Главная страница
- `task.md` - Описание задания
- `architecture.md` - Архитектура системы
- `interfaces.md` - Описание интерфейсов
- `installation.md` - Установка и запуск
- `models.md` - Модели данных
- `functionality.md` - Функциональность
- `conclusion.md` - Заключение
