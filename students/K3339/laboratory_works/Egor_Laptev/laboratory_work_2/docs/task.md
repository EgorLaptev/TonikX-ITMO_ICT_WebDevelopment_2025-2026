# Описание задания

## Индивидуальный вариант: Табло победителей автогонок

### Постановка задачи

Разработать веб-приложение для управления автогонками с функционалом табло победителей, регистрации участников и системы комментариев.

### Основные требования

#### 1. Табло победителей автогонок

Табло должно отображать информацию об участниках автогонок:

- **ФИО участника** - полное имя участника
- **Название команды** - команда, к которой принадлежит участник
- **Описание автомобиля** - информация об автомобиле участника
- **Описание участника** - дополнительная информация о гонщике
- **Опыт** - количество лет опыта участия в гонках
- **Класс участника** - категория участника (A, B, C)

#### 2. Просмотр автогонок и регистрация гонщиков

- Пользователь должен иметь возможность просматривать список всех гонок
- Пользователь может регистрироваться на гонки
- Пользователь должен иметь возможность **редактирования** своих регистраций
- Пользователь должен иметь возможность **удаления** своих регистраций

#### 3. Система комментариев и отзывов

- Пользователи могут оставлять комментарии к автогонкам
- **Предварительно комментатор должен зарегистрироваться** (иметь профиль участника)
- При добавлении комментариев сохраняются:
  - **Дата заезда** - дата проведения гонки
  - **Текст комментария** - содержание комментария
  - **Тип комментария** - категория (вопрос о сотрудничестве, вопрос о гонках, иное)
  - **Рейтинг** - оценка от 1 до 10
  - **Информация о комментаторе** - данные пользователя, оставившего комментарий

#### 4. Административная панель

- Администратор должен иметь возможность указания **времени заезда** и **результата** средствами Django-admin
- Удобный интерфейс для управления гонками и участниками

#### 5. Клиентская часть

- В клиентской части должна формироваться **таблица всех заездов и результатов** конкретной гонки
- Отображение полной информации о результатах

### Дополнительные требования

- Использование **PostgreSQL** для хранения данных
- Запуск PostgreSQL через **Docker**
- Система **аутентификации** (вход, регистрация, выход)
- Современный и удобный пользовательский интерфейс

### Реализованные функции
 Все требования из индивидуального варианта выполнены  
- Профили участников с управлением автомобилями
- Отдельная страница табло победителей
- Поиск по гонкам
- Пагинация списка гонок
- Система сообщений для пользователя
