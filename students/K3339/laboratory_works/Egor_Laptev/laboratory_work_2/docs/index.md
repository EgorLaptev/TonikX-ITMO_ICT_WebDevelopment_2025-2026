# Система управления автогонками - Racing Portal

## Введение

Данный проект представляет собой веб-приложение для управления автогонками, разработанное на основе Django Framework. Система позволяет пользователям регистрироваться на гонки, просматривать результаты, оставлять комментарии и управлять своими профилями участников.

## Основные возможности

- **Управление гонками** - создание и просмотр информации о гонках
- **Профили участников** - регистрация и управление профилями гонщиков
- **Управление автомобилями** - добавление и редактирование информации об автомобилях
- **Табло победителей** - отображение результатов гонок с полной информацией
- **Система комментариев** - возможность оставлять отзывы и комментарии к гонкам
- **Аутентификация** - регистрация, вход и выход из системы
- **Админ-панель** - управление данными через Django Admin

## Технологический стек

- **Backend:** Django 5.2.8
- **Database:** PostgreSQL 15
- **Frontend:** HTML, CSS (Tailwind CSS), Bootstrap 5
- **Containerization:** Docker, Docker Compose
- **Documentation:** MkDocs Material
